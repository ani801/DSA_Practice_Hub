# DSA_Practice_Hub
DSA Practice Hub is a full-stack MERN application designed to simplify and enhance your DSA preparation. It enables users to add, manage, and solve problems from any platform, while also tracking daily progress through a POTD (Problem of the Day) system and calendar visualization.

🚀 Features

✅ Add/Delete Problems with URL, title, topic, and difficulty

🌆 POTD (Problem of the Day): Random daily problem based on user preference

🕰️ Calendar Tracker: Track your daily solving streak visually

🔖 Topic & Difficulty Tagging: For better filtering and organization

🔍 Search and Filter through your problems

🌐 Fully responsive UI with a clean, modern design

🛠️ Tech Stack

Frontend : React.js,Tailwind CSS, React Router,React Icon.

Backend:  Node.js,Express.js,JWT (Authentication),Redis (caching)

Database: MongoDB,Mongoose

# Install dependencies for frontend

# Install dependencies for backend
